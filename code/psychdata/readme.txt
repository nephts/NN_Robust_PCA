Bevor die Klasse Psychdata aus psych.py benutzt werden kann, muss erst Pandas mit pip oder anaconda installiert werden.

Die BFI Daten enthalten auch angaben über Alter Geschlecht und  "education" (wahrscheinlich einfach der höchste Abschluss).
Die werden nicht berücksichtig bei der Berechnung der Korrelationsmatrix. 
Die Korrelationsmatrix kann mittels .get_corr() ausgegeben werden (siehe test.py)
